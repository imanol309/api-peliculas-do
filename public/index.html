Ruta para ver pelicula GET: /api/verPelis
Ruta para ver pelicula por el Id GET: /api/verPelis/:id
Ruta para ver pelicula por el nombre GET: /api/verPelis/titulo/:nombre
Ruta para modificar las toda la peliculas 'Necesitas un token autorizado' PUT: /api/modificarPelis/:id
Ruta para modificar un campo de la peliculas 'Necesitas un token autorizado' PUT: /api/modificarPelis/patch/:id
Ruta para crear una nueva pelicula 'Necesitas un token autorizado' POST: /api/crearPelis
Ruta para eliminar una pelicula de base de datos 'Necesitas un token autorizado' DELETE: /api/eliminarPelis/:id

Estas son las rutas para crearte un usuario para optener tu token autorizado

Ruta para ver los usuarios que estan logeados 'Necesitas un token autorizado' GET: /api/user/verUser
Ruta para ver los usuarios que estan logeados por Id 'Necesitas un token autorizado' GET: /api/user/verUserId/:id
Ruta para crear una nueva usuarios donde vas a tener tu token autorizado POST: api/user/crearUser
Ruta para ver si tu cuenta esta logeada POST: /api/user/loginUser
Ruta para eliminar un usuarios de la base de datos 'Necesitas un token autorizado' DELETE: /api/user/deleteUser/:id
